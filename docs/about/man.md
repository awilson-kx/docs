# Help for `man.q`


<i class="fab fa-github"></i>
[KxSystems/man](https://github.com/kxsystems/man)

## Examples
```txt
man "$"               / operator glyph
man "enum extend"     / operator name
man "read0"           / keyword
man ".z"              / namespace
man "-b"              / command-line option
man "\\b"             / system command
```

```txt
man "iterators"
man "kb"              / Knowledge Base
man "syscmds"         / system commands
man "wp"              / White Papers
```

## Arguments to `man`
```txt
man "--list"
```
